HTML5 : langage WEB d'organisation d'un contenu

J'ai commencé à m'initier à ce langage WEB en visionnant différentes vidéo Youtube, puis j'ai accéléré mon apprentissage en suivant la formation "Créez votre site web avec HTML5 et CCS3" d'Openclassrooms. Je profite aussi de Chat GPT pour obtenir très rapidement des explications et des fragments de code

CCS3 : langage WEB d'application de styles à un contenu

Je découvre le CSS très rapidement dans mes explorations du langage HTML. J'apprends, dans la même formation d'Openclassrooms, à appliquer des styles et des mises en forme à mon contenu et je découvre l'exigence de rigueur sur la syntaxe. Chat GPT est encore là pour m'assister dans mes résolutions de problèmes, il me permet d'aller à une vitesse à laquelle je n'aurai pas pu prétendre autrement. Il y a évidemment une autre expertise qui est tout aussi précieuse : l'expérience et le regard humains. Un ami développeur a passé mon code en revue, m'indiquant que mon code CSS comportait beaucoup de répétitions, et qu'il gagnerait en clarté et en maintenabilité si je le simplifiais. Ces corrections à venir font partie de mon plan d'action

JAVASCRIPT : un langage de programmation

Dans mes recherches sur le WEB, il est souvent désigné comme le troisième élément de la trilogie de langages du débutant. Il permet de réaliser des petites programmations, de rendre les pages dynamiques alors même que le processus de génération de page par le serveur est terminé. J'ai hâte de commencer à distiller ce genre de code, par exemple dans ce présent portfolio, mais je priorise en ce moment mon apprentissage sur les outils de développement "back-end"

Exemple de code JS bientôt disponible

PHP : la méthode OpenClassrooms m'a assez naturellement conduit à explorer ce langage pour mes premiers pas dans le développement back-end. J'installe Xampp pour que mon "client" se substitue à un serveur et je découvre que les variables peuvent stocker des types et des valeurs (integers pour les entiers, string pour les chaînes de caractères, float pour les décimaux, boolean pour les binaires, null pour... une valeur nulle). Je constate à nouveau l'exigeance de la syntaxe pour réussir des concaténations (assemblage de deux variables) et des interpolations (inclusion des variables dans une chaine) Je découvre les mécaniques des conditions (if, elseif, else) et des instructions. Dans le présent portfolio, je ne peux pas vraiment exploiter ce type de langage car GitHub Pages héberge des sites statiques. J'ai toutefois repéré que le système devrait pouvoir supporter une optimisation de mon code avec une fonction php de réécriture automatique des parties "répétées" de mes pages : par exemple l'entête (header), la barre de navigation, le pied de page (footer).

Python et SQL : j'ai découvert ces langages avec les exercices de candidature à la formation Simplon. Ce que j'avais déjà découvert en PHP m'a été utile, certaines notions étaient relativement transposables. A l'heure où vous lisez ce portfolio, c'est sur ces deux langages que je concentre mon temps d'auto-formation. J'ai installé Jupyter Notebook sur mon ordinateur, pour commencer à me familiariser au machine learning avec un exercice : le "titanic dataset" de Kaggle
